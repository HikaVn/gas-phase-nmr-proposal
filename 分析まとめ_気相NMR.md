# 冨宅喜代一氏の気相NMR研究 — 分析まとめ

作成日: 2026-06-13(Claude Code セッションでの調査・分析の要約)

## 1. 研究の概要

- 研究者: 冨宅喜代一(ふけ・きよかず)。神戸大学名誉教授 → 分子科学研究所(2012年装置移設)→ 豊田理化学研究所フェロー。
- 目的: 質量分析(MS)に構造解析機能を付与するため、質量選別した気相分子イオンの NMR 分光法を開発する。
- 中核原理: **磁気共鳴加速法**。高磁場(9.4 T)と低磁場(3.1 T)を傾斜磁場(最大 25 T/m)でつないだ専用超伝導磁石内の NMR セル(細長いイオントラップ)でイオン束を往復運動させ、両端通過時に πパルス RF で核スピンを反転。核磁気モーメントと傾斜磁場の相互作用力が往復毎に蓄積し、スピン状態の違いが飛行時間(TOF)の分裂(核スピン分極)として現れる。
  - 分裂幅: Δν = n(μN/Mν₀)(B_H−B_L)(2N+1)(n: プロトン数、M: 質量、ν₀: 初速、N: 往復回数)
  - 共鳴周波数を掃引しながら Δν を測定すると NMR スペクトルが得られる。
- 検出要件: 並進温度サブ mK(速度分布幅 <1 m/s、実績 0.4 m/s)、初速 ~100–264 m/s、5–6 往復。
- 要素技術: TS-PSW 減速器、進行波型多段減速器(TPW)、速度選別法、速度分散補償器(VDC)、超高真空対応 RF コイル+同調回路(400 MHz / 132 MHz)、磁場励起シーケンス。
- 到達点: トリエチルアミンイオン(TEA⁺)で 5 往復・5 回 RF 励起により TOF 幅約 10% の広がり(分極の兆候)。分子研の建物改修で中断し、決定的な原理実証は未完。ICR セルをイオン源とする構成の成立性はシミュレーションで確認済み。適用範囲の見積もりは分子量 ~500 まで。

## 4. 次のステップ(文献調査の計画)

①〜④について最新論文(目安 2020 年以降)からヒントを得る:

- ① 分子イオン冷却: 共同冷却(sympathetic cooling)、クライオバッファガス冷却、Stark/Zeeman 減速の進展
- ② 表面電場ノイズ: イオントラップ量子コンピュータ分野の patch potential / anomalous heating 対策
- ③ 少数イオン検出: 非破壊検出(image current)、単一イオン検出、量子増感
- ④ ロバストなスピン反転: 複合パルス、断熱パッセージ(quantum control / NMR 分野)
