1. 一言サマリー

これは何か: 質量分析(質量しか分からない)に、NMR(原子の化学環境を読む)を接ぎ木し、質量選別した気相イオン一種の「構造の指紋」を読む装置。
なぜ価値があるか: 「どの周波数(化学環境)に、何個の等価Hがあるか」というマップは構造そのもの。これを質量選別したイオンについて得られるのは、質量分析にも溶液NMRにも単独ではできない。

2. 原理を一歩ずつ

キモは3段階。① RFはスピンを反転させるだけ → ② 実際に加速するのは傾斜磁場(Stern-Gerlach力) → ③ 往復のたびに反転させ、力の向きを揃えて速度を蓄積。

ブランコを正しい位相で押し続けると振幅が育つのと同じ。RFは「押す力」そのものではなく「タイミングを合わせるスイッチ」。エネルギー収支を量子で見ると、高磁場コイルで光子 hνH を吸収、低磁場コイルで hνL を放出し、差し引き hνH − hνL ≈ 1.1 µeV/往復が運動エネルギーとして残る。

この 1往復あたり約 1 µeV というキックの小ささが、極低温(サブmK)冷却が必須な理由。熱的な速度ばらつきが 1 µeV より大きいと、この信号は埋もれてしまう。

3. 【体験】RFをかけると運動エネルギー分布はどうなるか

初期エネルギーが完全に一様(ノイズ0)の理想状態を仮定。RFをかけると、単一ピークが n+1 本の二項分布の多重線(コム)に分裂する。n(等価プロトン数)と N(往復回数)を動かして確かめよう。

4. 等価プロトン数 n とは(価数ではない)

NMRで「プロトン」= ¹H(水素)の原子核。n は「同じ磁気的環境(同じ共鳴周波数)を共有するH原子の個数」で、イオンの電荷(価数)とは無関係。

「環境」と「スピン状態」は別の軸。n 個の等価プロトンは同じ環境だが、各自のスピン(α/β)はバラバラでよい。このバラつきの合計(Mtotal)が n+1 本のコムを生む。

5. コムと包絡線

包絡線(ほうらくせん, envelope) = 離散ピーク(歯)の頂点をなぞる滑らかな輪郭線。櫛の歯と、歯先を結ぶシルエットの関係。

測れるものn は読めるか理由
分解した線(歯が見える)✅ 読める(偏極に無関係)線は既知間隔の整数位置。偏極は高さを変えるだけで位置は不変
未分解の包絡線の幅だけ❌ 曖昧幅 = f(n, 偏極)。両者が縺れて分離できない
TEA⁺で観測された「約10%の広がった単峰」は、歯が分解できず包絡線だけが見えた状態。理想の鋭いコムに対し、現実はボケた輪郭が見えただけ。これを分解することが、残る開発課題そのもの。

6. Q&A集(対話の歩み)

RFは100%運動エネルギーに変換される?量子論では?
いいえ。RFはまずスピン(ゼーマン)エネルギーになり、運動への変換は傾斜磁場が仲介する。正味でも、高磁場で吸収した hνH のうち KE に残るのは (BH−BL)/BH ≈ 67%、残りは低磁場コイルで再放出。実際は不完全πパルス等でさらに低い。なお「決定論的に100%反転する」(πパルスの忠実度)は別概念で、こちらが設計目標。
理想的なπパルスは、ランダムなスピンを同じ向きに揃える?
いいえ。揃える(整列・偏極)のではなく、全スピンをネガポジ反転(α↔β)するだけ。偏りの大きさ(|Mtotal|)は保たれ符号だけ反転する。可逆(ユニタリ)操作なので乱雑さは減らせない。「6α6β」のような半々(M=0)は反転しても半々のまま。
スピンのランダム性はRFの当たり方で決まる(モンテカルロ的)?
いいえ。理想πパルスは全プロトンを確実に反転させ、取りこぼしはない。乱数は反転の「前」、初期スピン状態にある(核スピンは生成時にほぼ50:50の無偏極=熱的な乱雑さ)。この初期の偏り Mtotal が二項分布になり、n+1 本のコムを作る。
スピンが全部揃っていたら、原理は成立しない?それとも好都合?
好都合。全イオンが同じ最大シフトを受け、1本の鋭いシフトピークになり感度が最大化(要因③を改善)。使えない理由は原理ではなく実装 ― 核スピンを揃えるのが極めて難しい(並進をサブmKに冷やしてもスピンは揃わない)。将来、気相イオンの超核偏極ができれば感度の切り札になる。
中途半端な偏極だと、分裂幅がわかっても n はわからない?
「包絡線の幅しか測れない」なら、その通り(幅=f(n,偏極)で縺れる)。だが「線を分解できる」なら、間隔(偏極に無依存の既知量)と線位置/本数から n は確定する。分解能が、偏極という余計な未知数を消す鍵 ― だから冷却(要因①)がここでも効く。
分裂幅は理論で正確に予測できる?
できる。Δν = n × (μp/Mν0)(BH−BL)(2N+1)。n 以外は全部既知(定数・質量はMSが与える・校正・計数)。だから順方向に予測でき、逆に分裂幅を測れば n が出る。質量 M が式に入るので、MSとのタンデムだからこそ定量計算が閉じる。
分子サイズが無限なら n も無限?
対称性(周期性)次第。無限でも周期的(ポリマー・結晶)なら n→∞(滑らかなガウス包絡線=連続に見える唯一の場面)。非周期的な巨大分子(タンパク質等)では n は小さいまま、代わりに周波数の種類が無限に増える(スペクトル混雑)。本装置が測る有限・非周期の気相イオンは常に後者側。

7. 起きやすい誤解(落とし穴)

誤解1: 「全状態が等確率」なら分布は平ら。→ 正: 微状態が等確率でも分布は平らにならず二項分布(中央が高い)。Mtotal=0 になる配置が一番多い(多重度)から。例: n=4 で M=0 は6通り、M=±2 は各1通り。
誤解2: n は価数のようなもの。→ 正: 価数(電荷)とは無関係。n は同じ環境のH原子の個数。TMA⁺は価数+1だが n=12。
誤解3: ランダム性はRFが確率的に当たるから。→ 正: 理想πは決定論的に全反転。乱数は反転前の初期スピン状態にある。
誤解4: n が大きいと連続スペクトルになり分析困難。→ 正: 1分子の n は有限・有限本数。間隔を決めるのは n でなく質量 M(∝1/M)。困るのは未分解時の幅の縮退と、大分子での多周波数の混雑。
誤解5: πパルスは「揃える」操作。→ 正: 「ネガポジ反転」操作。偏りは保たれ符号だけ反転。

8. 用語集

ゼーマンエネルギー
磁場中でスピンの向きにより生じるエネルギー差。9.4 Tの陽子で ΔE = hν = 1.66 µeV(= 400 MHz光子1個ぶん)。
Stern-Gerlach力
磁気モーメントが磁場勾配 dB/dz 中で受ける力 F = μ(dB/dz)。向きはスピンで逆になる。本手法で実際に加速する力。
πパルス(パイパルス)
スピンを正確に180°回す(反転する)コヒーレントなRFパルス。理想なら確率1で α↔β を入れ替える。
ラビ振動/忠実度
RF場でスピン状態が回転する現象。πパルスの「正確に反転できた度合い」が忠実度。N回の累積で効く。
核スピン分極(本手法の意味)
スピン状態の違いが飛行時間(TOF)の分裂として現れること。Δν として測る。
多重線(コム)
等価スピンが作る等間隔のピーク群。本数 n+1、強度はパスカルの三角形(二項)。「コム」は形の比喩で正式には多重線。
包絡線(envelope)
離散ピークの頂点をなぞる輪郭線。分解できないとこれだけが残り、n を曖昧にする。
Mtotal
等価プロトンの合計スピン射影 =(α数−β数)/2。各分子の加速量を決め、ピーク位置を決める。
分解度 R
R = 間隔 ÷ 線幅。R>2 でコムが分解できる。精度の本体。
VDC(速度分散補償器)
速い成分を減速・遅い成分を加速し速度分布を圧縮する装置。0.4 m/s(サブmK相当)を達成。
patch potential(浮遊電場)
電極表面の汚れ・仕事関数のムラが作る微小電位。µeV級イオンの軌道を乱す(要因②)。
